Crear la aplicación:
- ng new bases --standalone false --skip-git

Instalar las dependencias del package.json:
- npm install

Lanzar la aplicación:
- ng serve
- (con -o para abrir en navegador)

La app se lanza en 
- http://localhost:4200/

Usar ng para crear componentes:
- ng g component --help
- ng g c heroes/hero
- ng g c heroes/list

Usar ng para generar modulos:
- ng g m dbz
